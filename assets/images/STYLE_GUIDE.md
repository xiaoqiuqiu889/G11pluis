# 革命街 AI 原生重构 · 美术风格指南

> 本文档是 W5-A 美术补全产物的统一风格规范。
> 适用范围：`assets/images/artifacts/`（物件特写）、`assets/images/atmosphere/`（场景氛围近景）、`assets/images/redone/`（废弃图重做）。
> 美术基准：v6 canonical（`_legacy_v6/public/art-v5/`）10 张 + v6 V4 文档（`_legacy_v6/docs/V4-ART-DIRECTION-AND-SHOT-MANIFEST.md`）。

---

## 0. TL;DR（一句话锚点）

> **电影感、暗色调、暖黄高光、克制的现实主义伊朗与伊朗裔美国生活。**
> 不要做：温馨 / 明亮 / 卡通 / 赛博 / 装饰风。

| 维度 | 规范 |
|---|---|
| 整体调性 | 电影感（cinematic）、暗调、克制 |
| 画面语言 | 摄影写实（photorealistic），不是水彩手绘 |
| 比例 | 新建素材用 2.35:1（21:9）宽银幕；重做素材沿用 v5 的 16:9 |
| 颗粒 | 35mm 胶片颗粒（subtle） |
| 主光 | 单点暖色 practical 光源（灯泡、路灯、屏幕） |
| 辅光 | 冷色环境光（蓝灰、月光、晨曦） |
| 调色 | 低饱和暗色：橄榄绿、烟熏褐、暗琥珀、炭灰、vermillion 血橙点缀 |
| 文字 | 不出现可读文本（详见 §5） |

## 1. 色调规范

### 1.1 基底调色板

| 角色 | 颜色 | 用途 |
|---|---|---|
| **底色** | 炭灰 `#2A2825` ~ 烟褐 `#3A2F26` | 阴影、暗部、背景 |
| **暖光高光** | 烟熏琥珀 `#C9924A` ~ 灯泡黄 `#E8B461` | practical 光源、皮肤高光、物体受光面 |
| **冷光环境** | 暗蓝灰 `#4A5258` ~ 晨曦冷 `#6B7A8A` | 室外雨夜、机场、清真寺微光 |
| **点缀血橙** | vermillion `#C8462A` | 石榴籽、机场候机椅、雨衣、暖色衣物 |
| **墨绿陪衬** | 橄榄绿 `#5A6048` | 植被、外套、leila 头巾边缘 |
| **正色基准** | 漂白银白 `#D4C8B5` | 纸张、雨中反光、月光、桌面高光 |

> 永远不要出现：纯黑（会破坏电影感）、纯白（高调会破坏暗调）、饱和蓝（冷光应偏灰）。

### 1.2 三个场景的氛围色锚

| 场景 | 年代 | 关键色 | 光源 |
|---|---|---|---|
| `photo_lab_2008` | 德黑兰地下室 | 烟熏琥珀 + 暗木褐 | 单点 40W 灯泡（闪烁） |
| `farewell_2011` | 德黑兰国际机场 | 冷蓝灰 + 橙黄航班牌虚光 | 顶灯 + 远处暖灯 |
| `reunion_2024` | 伊斯坦布尔咖啡馆 | 暖琥珀（室内）+ 冷蓝灰（雨窗）| 台灯 + 窗外天光 |

## 2. 灯光规范

### 2.1 三点法则的电影简化

- **主光（key）**：来自一个 practical 光源——台灯、灯泡、屏幕、路灯。**永远不是死白闪光**。
- **辅光（fill）**：环境冷光（月光、晨曦、窗外天光）。色温 5000K-7000K。
- **轮廓光（rim）**：从主光对面的窗户 / 门口漏进来的冷光，勾出人物或物件边缘。

### 2.2 物件特写的灯光

物件特写必须用 practical 主光（不能让物件漂浮在抽象光里）：

- 毕业照：桌面 60cm 上方一盏 25W 暖灯泡
- 诗集：台灯斜 45° 投光
- 公交票：桌面油灯或远方煤油灯
- 行李牌：机场廊桥冷光 + 远处暖灯
- 石榴：头顶 25W 裸灯泡
- 折诗 / 信封：木桌台灯
- 鲁米诗集：咖啡馆木桌油灯

### 2.3 场景氛围近景的灯光

- photo_lab_2008 地下室：单点 40W 灯泡，闪烁（flicker 暗示）。
- farewell_2011 机场：顶灯冷光 + 远处航班信息板的暖橙虚光（**绝对不出现可读文字**）。
- reunion_2024 咖啡馆：木桌油灯 + 落地窗雨光。

## 3. 颗粒与质感规范

| 维度 | 规范 |
|---|---|
| 胶片颗粒 | 35mm 模拟颗粒，subtle，不要做 heavy grain |
| 锐度 | 中等偏柔（不是商业广告的 ultra sharp） |
| 暗角 | 轻微 vignette（V 1.05-1.10） |
| 后期调色 | 统一过一层 desaturate 10% + 暖阴影（teal-shadow 转 warm-shadow）|
| 画面瑕疵 | 接受：胶片划痕、灰尘颗粒、轻微 motion blur |

> 重新生成的 6 张废弃图（chapter-N.png）必须 **强制加同一层胶片颗粒 + 暗角 + 色调映射**，让 6 张图视觉上像同一卷胶片拍出来。

## 4. 比例与构图

### 4.1 比例

| 类别 | 比例 | 应用 |
|---|---|---|
| 物件特写（artifacts） | **2.35:1（21:9）** | 宽银幕感，让物件"漂浮"在两侧暗色留白 |
| 场景氛围（atmosphere） | **2.35:1（21:9）** | 视野更广，方便体现环境 |
| 重做的废弃图（redone） | **16:9** | 与 v5 canonical 一致，避免破坏既有视觉系统 |

### 4.2 构图法则

- **三分法**：主体放在画面 1/3 处（不是正中）
- **留暗**：画面至少 30% 是深色阴影区
- **前景虚化**：氛围近景要有 0.5-1m 的虚焦前景
- **depth of field**：物件特写用浅景深（f/2.0-2.8），氛围近景可用中景深
- **视线引导**：人物的目光 / 物件的高光引导观者看向画面深处理想信息

### 4.3 物件特写的取景

- 单个物件占画面 30-50%（不挤满）
- 物件周围留出 1/3 暗部
- 必带 1-2 件**次要环境元素**（一杯茶、一根烟、一块布）增加故事感

## 5. 文字与标识规范（红线）

### 5.1 绝对禁止

- ❌ 任何**可读的英文 / 中文**文字（subtitle、title、watermark、caption）
- ❌ 京东 / 任何合作方 logo
- ❌ 任何品牌 logo（即使是咖啡店、机场、商店）
- ❌ 任何 subtitle / 字幕
- ❌ 任何 UI 元素（按钮、菜单、对话框）
- ❌ 现代手机、电脑、互联网 logo

### 5.2 允许

- ✅ 装饰性 Persian / Arabic script（不作为内容出现，只作为纹理）
  - 例：诗集内页的 Rumi 排版（不可读为具体句子，但有视觉节奏）
  - 例：墙上 / 信封上的 Persian 书法（仅作纹理，不可读）
- ✅ 历史建筑 / 场所的固定 Persian 标识
  - 例：University of Tehran 大门上的 "دانشگاه تهران"（这是建筑铭牌，不是内容）
  - 例：机场地标、咖啡店门牌上的固定 Persian 词
- ✅ 时钟的数字（这是钟面设计，不是文字内容）
- ✅ 数字编号（如公交票 "304"）

### 5.3 文字风险检查清单（生成后必查）

- [ ] 屏幕 / 航班牌 / 招牌 是否有清晰文字？若是，必须 blurred / out-of-focus
- [ ] 书本 / 文件 是否有可读段文字？若是，缩小到装饰级别或改抽象纹理
- [ ] 商品 / 包装 / 糖包 是否有 logo？若是，去除
- [ ] 衣物 / 配饰 是否有文字？若是，去除
- [ ] 墙面 / 地面 是否有水印 / 涂鸦文字？若是，去除

## 6. 人物规范

### 6.1 角色锚点（来自 v6 V4 文档 + v5 canonical）

- **年轻莱拉（21-22 岁）**：椭圆脸、深色杏仁眼、浅灰头巾边缘露出松散卷黑发、深橄榄外套
- **中年莱拉（约 34 岁）**：保持相同脸型与眼睛，灰米色头巾、深橄榄外套，鬓边卷发与轻微年龄痕迹
- **年轻阿拉什（22-23 岁）**：瘦高、卷黑发、轻胡茬、深海军蓝夹克
- **中年阿拉什（约 35 岁）**：同样的卷发与瘦削轮廓，鬓角灰白、轻胡茬、炭灰长外套
- **卡姆兰**：短卷黑发、矩形眼镜、深色衬衫或连帽衫
- **玛丽亚姆**：深蓝头巾、棕色长外套；数学教师

### 6.2 人物真实感

- 写实手部（5 指、自然关节、可见血管）
- 自然皮肤（不磨皮、不完美、有雀斑、痣、细纹）
- 自然头发动态（不僵硬）
- 表情克制（不夸张、不戏剧化）
- 时代一致的服装（2008-2010 不出现 2015 后的款式）
- 永远不画**真实人物面孔**——生成 AI 风格插画，保留"角色锚点"但不锁定到任何真人

## 8. 后期统一处理清单

每张新生成的图，**生成后必须跑**以下后处理：

1. **分辨率检查**：1K（1280-1920px 宽）或 2K（2560-3840px 宽），高宽比严格按 §4.1
2. **裁切**：去除任何自动添加的水印、签名、AI 角标
3. **色调映射**：统一过 desaturate 10% + 暖阴影 + 轻微 vignette
4. **颗粒叠加**：35mm 胶片颗粒（强度 0.15-0.25）
5. **保存格式**：PNG 24-bit（不要 JPEG 压缩以保留细节）
6. **文件命名**：`{category}-{index}-{descriptor}.png`，全英文 + 数字
7. **路径**：
   - 物件：`assets/images/artifacts/{NN}-{descriptor}.png`
   - 氛围：`assets/images/atmosphere/{AN}-{scene}.png`
   - 重做：`assets/images/redone/{original-name}.png`

## 10. 验收 checklist

每张新生成的图在交付前必须通过：

- [ ] 比例正确（artifacts/atmosphere 2.35:1，redone 16:9）
- [ ] 整体暗调（无大面积白、无饱和蓝）
- [ ] 主光是 practical 光源（不是抽象光）
- [ ] 有 35mm 颗粒感
- [ ] 物件特写有 1-2 件次要环境元素
- [ ] 场景氛围近景有 0.5-1m 虚焦前景
- [ ] 人物（如有）符合角色锚点
- [ ] 无可读文字 / 无 logo / 无品牌标识
- [ ] 无现代互联网元素
- [ ] 色调与 v5 canonical 一致
- [ ] 文件命名符合 §8 规范

## 11. 风格速记卡（给后续 AI 美术 agent 用）

```
cinematic, dark, photorealistic, low-saturation,
olive + brown + charcoal + smoky amber + cool blue-grey + vermillion accent,
35mm film grain, subtle vignette,
single warm practical light + cool ambient,
2.35:1 widescreen (or 16:9 for canonical),
realistic Iranian / Iranian-American 2008-2024,
no readable text, no logos, no brand markings,
no cute, no bright, no cartoon, no cyberpunk,
preserve character anchors, natural hands and faces,
muted, restrained, bittersweet, cinematic,
always autumn or winter, wet or dry, never summer-bright
```
