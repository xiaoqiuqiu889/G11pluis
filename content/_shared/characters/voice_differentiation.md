# 跨案人物口胞区分说明（UP-20260715-026 / W12-UP-026）

## 1. 口胞区分核心原则（评审推荐）

> **natasha 偏"苏式教育下的克制 + 偶尔的感性"**
> **ilya 偏"维也纳式的理性 + 偶尔的浪漫"**
>
> 让两案口胞明显不同，避免"两案一个调"—— 失去第二案的独特性

## 2. natasha 句式风格（与 leila 区分）

### 2.1 苏式教育下的克制

| 维度 | leila（case_01）| natasha（case_02）|
|---|---|---|
| 句式长度 | 短句 + 偶尔长句（德黑兰口语节奏）| 短句为主 + 偶发 1 个长句（俄罗斯文学节奏）|
| 情感表达 | 直接 + 偶尔用"我心里知道"暗示 | 几乎不用第一人称感叹；用物理意象代替 |
| 物理意象 | 灯泡 / 胶片 / 工具盒 | 延音踏板 / Petrof 大提琴 / 松香渍 / 中央 C 键 |
| 时代符号 | 16mm 放映机 / 鲁米诗集 / 公交票 | 肖斯塔科维奇 Op.38 / Yamaha U3 / 红色笔记本 |
| 句尾标点 | 句号 / 偶尔问号 | 句号 / 偶尔省略号（俄罗斯文学翻译腔）|

### 2.2 偶尔的感性（在收束场景触发）

| 场景 | leila 感性方式 | natasha 感性方式 |
|---|---|---|
| 离别 | "我本来想叫住你……" | "5:55 我在 4 秒后才接——和你对视也是 4 秒" |
| 重逢 | "你把那张照片带了多少年？" | "我在 1985 总谱上签 И. Б.。我把那一页撕下来带了 19 年" |
| 分手 | "绿灯亮起"（无言）| "21:05 红灯变绿——我们在同一刻回头笑一下" |

**关键差异**：natasha 的感性**比 leila 更内敛**——19 年的等待通过"4 秒沉默"和"同一刻回头笑一下"表达，**不直接说"我爱你"或"我想你"**。

## 3. ilya 句式风格（与 arash 区分）

### 3.1 维也纳式的理性

| 维度 | arash（case_01）| ilya（case_02）|
|---|---|---|
| 句式长度 | 中等长度（工科系）| 短句 + 偶发德文语法倒装（维也纳德语训练）|
| 情感表达 | 工具盒 + 工具动词代替情感（"我把那张折诗收进工具盒"）| 铅笔圈注 + 笔记本代替情感（"我在总谱上签 И. Б.。我把那一页撕下来"）|
| 物理意象 | 螺丝刀 / 备用灯泡 / 胶带 | 铅笔 / 红色笔记本 / Aeroflot 标签 |
| 时代符号 | 16mm 放映机维修 / 鲁米诗集 | 肖斯塔科维奇 Op.38 / Bösendorfer Imperator |
| 句尾标点 | 句号 / 偶尔"……" | 句号 / 偶尔"——"（em-dash，俄罗斯文学翻译腔）|

### 3.2 偶尔的浪漫（在收束场景触发）

| 场景 | arash 浪漫方式 | ilya 浪漫方式 |
|---|---|---|
| 离别 | "我在机场外站了 13 年……" | "B 面 23:14 处你那个大提琴长音——莉莎听过 6 次" |
| 重逢 | "我带了你 2008 那张同版照片……" | "305 琴房的中央 C 键——你的 Petrof 大提琴松香渍还在那" |
| 分手 | "咖啡馆外的路口"（无言）| "我打电话给莉莎——'今晚云不遮德意志歌剧院的屋顶'" |

**关键差异**：ilya 的浪漫**比 arash 更克制**——通过"莉莎听过 6 次"和"莉莎的电话" 表达，**不直接说"我记得你"或"我一直想你"**。
