- **现状**：12 动作按钮 5×3 网格平铺，玩家第一次接触会"按钮恐惧"—— 信息密度过高

  2. 其他 11 个半透明 + 灰度
  3. 鼠标悬停才亮起 + 显示按钮名
  4. 玩家按过一次之后，所有按钮恢复全亮（已熟悉的引导不重复）
- **权衡**：会增加 UI 复杂度，但能让"12 行为"从"产品意图"变成"玩家可用的工具"
