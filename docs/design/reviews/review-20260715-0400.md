## 4. Step 6：体验升级迭代 ⭐

### 6.1 玩家同理心审视
- **玩家第一次进入真实 AI 原生模式（不是 mock）会发生什么？** W4 部署后，真实 LLM 接入，但 W3-A2 报告："AI 工具会强制添加 vocal"，**音色质量妥协**（sine-wave vs 真实乐器）。真实 LLM 反应 + sine-wave 模拟乐器 = **声音哲学"留白多、不抢台词"达成，但"伊朗艺术电影配乐感"未达成**。
- **玩家通关第二案（A《莫斯科没有童话》）的体验是什么？** W6 第二案有完整内容但 0 张图 0 音频——**二案"可玩但没灵魂"**，玩家只能看文字。

### 6.2 策划批判性审视
- **5 句备选台词的 selection_rule 用 referencedMemoryIds 自动匹配**——这意味着 LLM 必须先识别玩家 2008 行为记忆 → 选 1 句台词。**但 LLM 的选择正确率是多少？** 没有 5 句备选是为了"内容丰富"，还是"AI 选择能力有限"？**当前没有 A/B 数据**。
- **farewell_2011 没有 5 句备选**——W4-Content-Update 报告说"UP 未直接要求"——但 5 句备选作为设计模式应该跨场景统一。**farewell_2011 的 2 个 mandatory echo 同样需要备选台词**。
- **私人终章 3 段是模板化还是真生成？** 当前实现 3 sections 引用了 mandatory_echoes + bus_ticket 故事，但**如果玩家没有触发特定 echo，私人终章是"通用 3 段"还是"省略触发 echo 的段落"？**

### 6.3 商业化心理审查
- **¥48 收藏版现在值了**——通行证 + 双视角 + 原声 + 私人终章。**但 3 个 short recap（1995 莱拉 / 阿拉什修望远镜 / 玛兹雅）是周度推送（决策 5 验收 D7 召回），不是收藏版权益**——**¥48 收藏版 vs 通行证 200 积分 vs 短案权益** 边界仍模糊。
- **BYOK 玩家与免费玩家的体验差**——BYOK 玩家消耗自己 token，无 L3 降级风险；免费玩家 150 调用后强制降级。**这是合理的产品分级，但需在 UI 透明告知**。

### 6.4 母题与物件深度审视
- **第一案物件 8 件 vs 第二案物件 12 件**（差距 +4 件）——第二案多了"中介者物件"（1986 巧克力锡纸、walkman + 磁带、明信片）。**这 12 件物件在 3 个场景合同里都有独立母题吗？** W6 子 agent 报告里"6 mandatory echoes + 14 causal_seeds"——但**物件 vs 母题 vs 因果种子** 的对应关系需要验收。
- **"别松手" / "两张相同照片" / "我到了" 第一案跨章母题在 reunion_2024 已升级（握—松三跳链 + 照片 + "我到了"）**——但**第二案有什么跨章母题？** W6 没明说。**W6 跨章母题缺失** = V5 命题"内容可规模化"只证明"工程复用"，没证明"母题可规模化"。
